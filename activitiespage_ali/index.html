大赛简介

互联网时代下无处不在云计算，各行各业的交错连接让DT时代更加丰富，在“互联网、创新、创业”的大背景下，阿里云为开发者们准备了一场技术角逐。

编程马拉松，强调浓烈的开发热情和对技术的不断探索的极客精神。阿里云致力于为开发者助力，希望通过全面开放 API/SDK 让云端开发有更多想象空间和实施的机会，同时还有丰厚的奖金和 “2015杭州·云栖大会”的线下路演机会。 同时，大赛还设置了基于阿里云安全API的专门赛区，同样提供了一个专项大奖，旨在让我们具备开发能力、能理解基本安全攻防原理的安全爱好者们能够一起共建云生态的安全。

你们只管专注于编码，其他的阿里云来护航！

项目规则

开发团队须选择并使用以下阿里云 API/SDK，在限定的时间内完成开发。如选择对应的的API，可以加入该 api 所设立的QQ群进行沟通交流，并可以向技术负责人进行咨询或寻求帮助，所有阿里云的公开API和SDK均可以使用，包括：OSS、ODPS、OCS、CDN、OpenSearch、KVStore、ACE以及安全相关的 API等。

答疑交流：362958892 (QQ群)

阿里云 API/SDK：

查看阿里云API 查看阿里云安全API

开发项目的要求：

  • 代码实现功能完整，有完整的单元测试，能编译、测试、运行通过
  • 代码工整，有统一的代码风格
  • 代码遵循Apache 2.0的开源协议进行开源
  • 如果发现剽窃已有的开源代码，参赛者将失去参赛资格
  • 该开源项目的作者、主要贡献者例外，也就是已有的开源项目也可以报名参加
  • 参赛者比赛项目代码须托管指定代码托管平台：Git@OSC , 提供作品功能的简要说明

项目开发参考方向

    • SDK的语言补充类型
    • 开源软件与阿里云产品的整合
    • 通过API/SDK的封装， 提供方便易用的工具
    • 阿里云安全 API

提供阿里云官方以外的语言类SDK，比如Go或者Ruby等语言类型的SDK 也可以对现有的阿里云的SDK进行完善扩展
